Diffusion（2）-开山作DULN的详细推导

了解了Diffusion大概的研究现状后，我只是大概明白了Diffusion的流程和框架，对于怎样去具体的实现它还没有认识。所以我首先去读了Stanford的那篇首次将Diffusion应用到机器学习的文章：Deep Unsupervised Learning using Nonequilibrium Thermodynamics。并对里面的公式进行了详细推导和整理，如下（滑动可翻页）：

总的来说Diffuion就是一个学习高斯核，也就是学习一些列高斯分布的参数（均值和方差）的过程。我们的训练目标是 找到模型的最大对数似然。而可以用数学方法证明这个对数似然有下界，切在最理想情况下对数似然就等于这个下界。于是我们的训练目标就变成了最大化这个下界（文中用\(K\)表示）。利用数学方法和一些近似，成功的将下界化为了KL散度和熵的和的形式。

在真正进行训练的时候，由于采样这个操作无法进行反向传播，所以使用了一种变通的方法。还经过推到，实现了已知初始条件就可以算出任意时刻t的状态而不用一步一步用循环去计算。

不足之处在于作者使用的代码很老，难以复现。此外，这个版本的Diffusion效果很差（相比于今天）。所以没有对这个工作进行复现。
